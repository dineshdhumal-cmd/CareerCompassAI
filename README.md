# Career Compass AI

Career Compass AI helps students discover alternative career routes when their original plans don't go as expected.

## Problem

Many students face setbacks such as:
- Exam failures
- Low scores
- Placement rejections
- Career confusion

Most platforms tell students what careers exist.

Career Compass AI answers:

"What should I do when my original plan doesn't work?"

## Features

- AI-powered route recalculation
- Alternative career pathway generation
- Personalized guidance
- Perspective and motivation engine
- Multi-screen career exploration workflow

## Tech Stack

- Python
- Flask
- Google Gemini AI
- HTML
- CSS
- Bootstrap
